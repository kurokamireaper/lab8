# Онлайн-планувальник подорожей

Онлайн-планувальник подорожей — це веб-застосунок, який допомагає користувачам планувати свої поїздки в одному місці: маршрути, дні подорожі, бюджет, житло та активності.

## Яку проблему вирішує

- Немає потреби тримати нотатки в різних додатках, табличках і месенджерах.
- Уся інформація про подорож (маршрут, витрати, бронювання) зібрана в одному зручному інтерфейсі.
- Легко поділитися планом з друзями або родиною.

## Основні функції

- Створення подорожі (місто, країна, дати, опис).
- Планування маршруту по днях (перегляд у вигляді списку або таймлайна).
- Додавання місць (пам’ятки, кафе, готелі) до кожного дня.
- Орієнтовний бюджет подорожі (квитки, житло, харчування, розваги).
- Збереження важливих нотаток (контакти, бронювання, чеки).
- Можливість поділитися посиланням на подорож з іншими користувачами.

## Для кого цей продукт

- Студенти, які планують подорожі з друзями.
- Мандрівники, які люблять все структуровано організовувати.
- Родини, які хочуть чітко бачити маршрут і бюджет поїздки.
- Фрилансери/remote-працівники, що комбінують роботу та подорожі.
