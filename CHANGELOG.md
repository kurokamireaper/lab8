# Журнал змін — Онлайн-планувальник подорожей

Формат: **[версія] – дата** та короткий опис змін у проєкті.

## [1.0.0] - 2025-03-15

### Додано
- Повноцінний функціонал входу та реєстрації користувача.
- Сторінку списку подорожей з кнопкою **"Нова подорож"**.
- Сторінку деталей подорожі з відображенням днів і активностей.
- Вкладку **"Бюджет"** з підрахунком загальної суми витрат.

### Виправлено
- Покращено відображення дат для різних часових поясів.
- Уніфіковано шрифти та відступи на основних сторінках застосунку.

---

## [0.3.0] - 2025-03-10

### Додано
- Чернетку REST API для роботи з подорожами:
  - `GET /api/trips` — список подорожей.
  - `POST /api/trips` — створення нової подорожі.
  - `GET /api/trips/{id}` — деталі конкретної подорожі.
- Поле “Орієнтовний бюджет” при створенні подорожі.
- Відображення бюджету в картці подорожі на головній сторінці.

### Змінено
- Оновлено структуру сторінки деталей подорожі для кращої читабельності.
- Невеликі покращення адаптивної верстки (мобільна версія).

---

## [0.2.0] - 2025-03-05

### Додано
- План дня з можливістю додавати активності (час, тип, назва, орієнтовна вартість).
- Навігацію між днями однієї подорожі.
- Редагування назви подорожі, опису та дат поїздки.

### Виправлено
- Дрібні візуальні баги на мобільних пристроях.
- Некоректне вирівнювання кнопок на головній сторінці.
