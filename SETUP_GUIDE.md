# Посібник з установлення та запуску

Цей документ описує базові кроки для запуску онлайн-планувальника подорожей у вигляді веб-застосунку (демо-варіант).

---

## 1. Необхідні інструменти

Для запуску демо-версії достатньо:

- Будь-який сучасний браузер (Chrome, Firefox, Edge, Safari).
- Підключення до інтернету.

Для локального запуску (як розробник):

- Встановлений **Node.js** (версія 18.x або новіша).
- **Git** (для клонування репозиторію).
- Текстовий редактор (VS Code, WebStorm або інший).

---

## 2. Запуск як звичайний користувач

1. Перейдіть за посиланням на продакшн/демо-сайт (коли буде розгорнуто).
2. Дочекайтеся завантаження головної сторінки.
3. Зареєструйтесь або увійдіть у систему.
4. Почніть створювати свою першу подорож.

> У межах лабораторної роботи URL може бути умовним, наприклад:  
> `https://travel-planner-demo.example.com`

---
