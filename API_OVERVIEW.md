# Огляд API: Онлайн-планувальник подорожей

Цей документ описує вигаданий приклад REST API для онлайн-планувальника подорожей.  
API дозволяє:
- отримувати список подорожей;
- створювати нову подорож;
- переглядати деталі однієї подорожі.

> Це навчальний приклад для лабораторної роботи. Реального сервера може не бути.
